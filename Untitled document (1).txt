🏷️ Project Title
AI SEO Blog & Content Cluster Generator for Business Websites – Glow & Grace Salon


📌 Project Overview
This project demonstrates how Prompt Engineering can be used to generate SEO-optimized blog content and structured content clusters for business websites.
Many local businesses fail to attract traffic because their content does not match search intent, lacks structure, and is not optimized for search engines. This project solves that problem by creating a structured AI prompt system that generates ranking-focused blog content.
The approach follows real-world SEO strategies used by agencies, where content is built as clusters instead of random blogs.
🎯 Objective
The objective of this project is to design a reusable prompt workflow that can:
* Generate SEO-optimized blog outlines (H1–H3 structure)
* Write long-form blog content targeting search intent
* Create content clusters (pillar blog + supporting blogs)
* Adapt content for local SEO (city + service keywords)
* Support business goals like leads, trust, and authority


🧠 Prompt Framework
Act as an SEO content strategist and blog writer.
Create SEO-optimized blog content for a local business.
Business Details:
* Business Name: [Enter Name]
* Location: [City]
* Business Type: [Salon / Clinic / Agency]
* Target Audience: [Who they serve]


Generate the following:
1. Pillar Blog:
* SEO title with keyword
* H1, H2, H3 structure
* Detailed long-form content (800–1200 words)
* Include local keywords naturally
2. Supporting Blogs:
* 3–5 blog ideas related to the main topic
* Short blog content (300–500 words each)
3. SEO Strategy:
* Primary keyword
* Supporting keywords
* Search intent (informational / transactional)
4. Internal Linking:
* Suggest how blogs should link to each other
Rules:
* Keep content human, clear, and helpful
* Avoid keyword stuffing
* Focus on solving user problems
* Write content ready for publishing
