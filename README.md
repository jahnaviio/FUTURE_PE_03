## 📌 Project Overview
This project demonstrates how Prompt Engineering can be used to create SEO-optimized blog content and structured content clusters for local businesses.

Instead of writing random blogs, this system generates:
- A pillar blog (main topic)
- Supporting blogs (related topics)
- SEO keyword strategy
- Internal linking structure

---

## 🎯 Objective
To design a reusable prompt system that can:
- Generate SEO-friendly blog outlines (H1–H3)
- Create long-form blog content
- Build content clusters (pillar + supporting blogs)
- Adapt content for local SEO (city + service)
